C 语言进阶指南

前言

本文章主要面向的是已经基本熟悉 C 语言的初学者，而目的则在于通过对 C 语言具体机制的介绍，使得读者可以理解 C 的本质，面对问题时，能够通过 C 的思想而非强制记忆的规则得出答案。

所采取的方式是，通过对某个语言机制存在的原因，来理解这个特性为何这样设计，以及推测在其他情景下这个特性会如何表现。可以的时候，会结合一些例子提出问题。
